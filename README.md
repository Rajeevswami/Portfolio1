# Rajeev Swami — Portfolio

Django Full Stack Developer Portfolio Website

## 📁 Folder Structure

```
rajeev-portfolio/
│
├── index.html                 ← Main HTML file (open this)
├── README.md
│
└── assets/
    ├── css/
    │   └── style.css          ← All styles
    ├── js/
    │   └── main.js            ← All interactions
    └── img/
        └── photo.jpg          ← Your profile photo
```

## 🚀 How to Open

Just double-click **index.html** — opens in any browser.

## 🌐 How to Deploy FREE

### Option 1 — Netlify (Easiest)
1. Go to netlify.com
2. Drag the entire `rajeev-portfolio` folder onto the dashboard
3. Done! Live link in 30 seconds

### Option 2 — GitHub Pages
1. Create repo named: `Rajeevswami.github.io`
2. Upload all files
3. Settings → Pages → Deploy from main branch

## ✏️ What to Update

| What | Where |
|---|---|
| Project GitHub links | `index.html` → pbt buttons |
| Live demo links | `index.html` → pbt.live buttons |
| Skill percentages | `index.html` → `data-w` attributes |
